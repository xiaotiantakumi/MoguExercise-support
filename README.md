Mogu Exercise is an iOS AR-based chewing fitness training app that uses ARKit's facial tracking to detect chewing motions in real-time. The app gamifies the chewing experience with character evolution, combo systems, and visual feedback.

**Features:**
- Real-time chewing detection using ARKit face tracking
- Meal session tracking and statistics
- Daily/weekly/monthly chewing history graphs
- Character evolution and gamification
