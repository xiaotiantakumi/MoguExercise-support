## About Mogu Exercise

Mogu Exercise is an iOS AR-based chewing fitness training app that uses ARKit's facial tracking to detect chewing motions in real-time.

### Key Features

- **Real-time Chewing Detection** - Uses ARKit face tracking to count chews
- **Meal Session Tracking** - Track eating habits and statistics
- **Gamification** - Character evolution and combo systems
- **Privacy First** - All data processed locally on your device
