### Common Issues

#### AR Not Working
- **Problem**: Camera isn't tracking your face
- **Solution**: 
  - Ensure you're using a device with a TrueDepth camera (iPhone X or later)
  - Check that camera permissions are granted
  - Make sure you're in a well-lit environment
  - Hold the device at arm's length with your face clearly visible

#### Chewing Not Detected
- **Problem**: App isn't counting your chews
- **Solution**:
  - Position camera so your entire face is visible
  - Chew normally - the app detects jaw movement
  - Ensure AR tracking is active (look for AR indicator)
  - Try restarting the AR session in app settings

#### App Crashes
- **Problem**: App unexpectedly closes
- **Solution**:
  - Restart your device
  - Ensure you're running the latest iOS version
  - Check for app updates in App Store
  - If the problem persists, please [report a bug](#reporting-bugs)

## üêõ Reporting Bugs

Found a bug? We'd like to know about it!

### Before Reporting

Please check:
1. You're using a supported device
2. You're running the latest iOS version
3. You have the latest app version
4. The issue hasn't already been [reported](https://github.com/xiaotiantakumi/MoguExercise-support/issues)

### How to Report

**Option 1: GitHub Issues (Recommended)**
- Click [here](https://github.com/xiaotiantakumi/MoguExercise-support/issues/new?template=bug_report.yml) to report a bug
- Fill out the bug report template with:
  - Device model and iOS version
  - App version
  - Steps to reproduce
  - Expected vs actual behavior
  - Screenshots (if applicable)

**Option 2: Email**
- Send details to: **xiaotiantakumi@gmail.com**
- Include "Mogu Exercise Bug" in the subject line
